How to use
Given that all the modules are installed,all you need to do is run the code in jupyter notebook

1.The dashboard
Once the application opens, you will be greeted with a dashboard,wherein you can search the ticker of the stock you want to analyze.Beside the search bar you will find a drop down list from which you can choose the time window to analyze historical data

2.Stock analysis
Here you can analyze the stock you had searched for via graphs.This tab also displays key metrics like current price,openprice,Beta,Sector,etc.You can add stocks to your watchlist here

3.Portfolio Comparison
Here you can compare different stocks(very necessary before investing in a stock)

4.Excel reports
From here you can download an excel report of the stock that you had previously searched for(Nothing extraordinary,just a brief report)

5.Watchlist and history
Here,you will find your watchlist and the history of your activity on this app
